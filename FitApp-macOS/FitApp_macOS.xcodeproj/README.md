# Xcode Project Placeholder

This repo includes source scaffolding for macOS + watchOS.

If this folder does not yet contain a generated `project.pbxproj`, create the Xcode project locally:

1. In Xcode, create a new App project named `FitApp_macOS` (SwiftUI, Swift).
2. Add a watchOS companion app target.
3. Map source groups to these folders:
   - `FitApp-macOS/Models`
   - `FitApp-macOS/Views`
   - `FitApp-macOS/Services`
   - `FitApp-watchOS/Models`
   - `FitApp-watchOS/Views`
   - `FitApp-watchOS/Services`
